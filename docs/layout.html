Layout

Almost all front-end frameworks provide ways to design a layout. Subtleframe includes some basic elements to create layouts for Dashboards and Deports.

Grid

The simplest way to create a layout for a full screen dashboard is with the help of a grid. Subtleframe includes a CSS Class called grid that uses the property display: grid. Grids are 12x12 arrangements. You can place a div inside a grid setting the grid-area CSS property. See the following example:

Example

1 / 1 / 5 / 5

5 / 1 / 13 / 5

1 / 9 / 13 / 13

<div class="grid_layout">
  <div style="grid-area: 1 / 1 / 5 / 5; background-color: rgba(0,0,0,0.1); text-align: center;">
    <p>1 / 1 / 5 / 5</p>
  </div>
  <div style="grid-area: 5 / 1 / 13 / 5; background-color: rgba(0,0,0,0.1); text-align: center;">
    <p>5 / 1 / 13 / 5</p>
  </div>
  <div style="grid-area: 1 / 9 / 13 / 13; background-color: rgba(0,0,0,0.1); text-align: center;">
     <p>1 / 9 / 13 / 13</p>
  </div>
</div>

The grid-area property places each panel by it's coordinates on the 12x12 grid (taking 1 as the first row / column and 13 as the last):

grid-area: first-row / first-column / last-row / last-column

To learn more about grids you can visit this link

Panels

A div inside a grid layout can become a panel by adding the class gpanel to it. A panel is simply decorated to add some padding, color and rounded borders

You can also use the gtitle class to create a title inside a div in a grid.

Example

Panel 1

1 / 1 / 5 / 5

Panel 2

5 / 1 / 13 / 5

Panel 3

1 / 9 / 13 / 13

<div class="grid_layout">
  <div class="gpanel" style="grid-area: 1 / 1 / 5 / 5;">
    <div class="gtitle">Panel 1</div>
    <p>1 / 1 / 5 / 5</p>
  </div>
  <div class="gpanel" style="grid-area: 5 / 1 / 13 / 5;">
    <div class="gtitle">Panel 2</div>
    <p>5 / 1 / 13 / 5</p>
  </div>
  <div class="gpanel" style="grid-area: 1 / 9 / 13 / 13;">
    <div class="gtitle">Panel 3</div>
    <p>1 / 9 / 13 / 13</p>
  </div>
</div>

Horizontal and vertical layout

Inside grids you usually stack components vertically or horizontally. You can use the h_layout and v_layout CSS Classes to create a simple stacked layout. Divs placed inside any of these layouts will automatically resize and stack inside their parent.

Tabs

Subtleframe Tabs use javascript to create a tabbed layout to hide and show different divs.

Example 1

Tab 1
Tab 2
Tab 3
<div class="gpanel" id="tablayout1" style="height: 200px; width: 100%;" >
    <div id="tab11"> Tab 1 </div>
    <div id="tab12"> Tab 2 </div>
    <div id="tab13"> Tab 3 </div>
</div>
var tl = new Subtle.TabsLayout("tablayout1", [
    {div:"tab11", icon:"<i class='icon-sparkline'></i>", label:"ONE"},
    {div:"tab12", icon:"<i class='icon-layout-1'></i>", label:"TWO"},
    {div:"tab13", icon:"<i class='icon-warehouse-2'></i>", label:"THREE"},
],{position:"center", show_labels_inline: false, icon_size: 20});

Constructor and Params

To create a Tabs Layout, use the constructor:

Subtle.TabsLayout(object_id, tabs, params)

| Where object_id is the id of the parent container and tabs is a list of objects | with details on each tab: {div: div_id, icon: html_code_for_icon, label: labe_text}. | You can further edit a tabs layout with the following params:

paramtypedefaultdescription
show_labelsBooleantrueShow / hide labels
show_labels_inlineBooleanfalseShow labels on the same line as icons
show_iconsBooleantrueShow / hide icons
icon_sizeNumber16Icon size (pixels)
positionString ("center", "left", "fill")"center"Position of the tabs
directionString ("horizontal", "vertical")"horizontal"The orientation of the tabs
dock_bottomBooleanfalsePlace the tabs on the bottom of the container (for horizontal direction)
colorString (from Subtle.COLORS, hex or rgb)COLORS['primary']Selected tab color
