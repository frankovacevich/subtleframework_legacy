Colors

Subtleframe provides a way to change the main colors of the web app through javascript. Other colors can be changed overriding the css styles. It's best if you set the colors you'll be using on javascript before you create any control. If you change colors on runtime in your application, remember to update your controls to see the effects.

Dark and light mode

The main colors used are stored on two javascript objects:Subtle.LIGHT_MODE andSubtle.DARK_MODE. By default, Subtleframe uses the colors on the light mode, but you can switch to dark mode with the function:

Subtle.SET_COLOR_MODE(mode = "dark")

Remember to call this function on the first part of your script for best performance.

You can useSubtle.SET_COLOR(color, key = "primary") to edit a color.
