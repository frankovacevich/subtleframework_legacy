Plotly helpers

Plotly is a free and open source graphing library that is easy to use and integrate on any web app. Subtle framework provides a bunch of methods for quickly drawing charts with plotly.

Use the Subtle.PlotlyObject(object, params) to create a plotly helper.

Use the method Subtle.PlotlyObject.draw() to draw the plot after adding the traces you want.

To remove all traces from the plotly object, use Subtle.PlotlyObject.traces = []. This will remove all current traces after you redraw the object.

Params

Scatter plot

Use Subtle.PlotlyObject.add_scatter(data, params) to add a scatter plot to a plotly object. 

var plotly_scatter = new PlotlyObject("plotly_scatter", {});
plotly_scatter.add_scatter([[1,2,3,4], [7,5,6,8]], {show_markers: true});
plotly_scatter.draw();
paramtypedefaultdescription
nameString""Series name (used for the legend)
colorString (from Subtle.COLORS, hex or rgb)COLORS['primary']Plot color
show_markersBooleanfalseShow/Hide markers
show_linesBooleantrueShow/hide lines

marker_symbolString ("circle", "square", "diamond", "cross", "x", "triangle-up", "triangle-down")"circle"Marker symbol (more)
marker_sizeNumber6Marker size
zindexNumber0Index to place the plot above or below other traces.

Column chart

Use Subtle.PlotlyObject.add_columnchart(data, params) to add a column chart to a plotly object. 

var plotly_column = new Subtle.PlotlyObject("plotly_column", {});
plotly_column.add_columnchart([["Alpha", "Beta", "Gamma", "Delta"], [7,5,6,8]], {name: "X"});
plotly_column.add_columnchart([["Alpha", "Beta", "Gamma", "Delta"], [3,7,8,8]], {name: "Y"});
plotly_column.add_columnchart([["Alpha", "Beta", "Gamma", "Delta"], [6,6,6,25]], {name: "Z"});
plotly_column.draw();
paramtypedefaultdescription
nameString""Series name (used for the legend)
colorString (from Subtle.COLORS, hex or rgb)COLORS['primary']Plot color
zindexNumber0Index to place the plot above or below other traces.

To create a stacked column chart, you could use plotly_column.layout.barmode = "stack".

Pie chart

Use Subtle.PlotlyObject.add_piechart(data, params) to add a pie chart to a plotly object. 

var plotly_pie = new Subtle.PlotlyObject("plotly_pie", {});
plotly_pie.add_piechart([["Alpha", "Beta", "Gamma", "Delta"], [7,5,6,8]]);
plotly_pie.add_piechart([["A", "B", "C", "D"], [3,7,8,8]]);
plotly_pie.add_piechart([["X", "Y", "Z", "W"], [6,6,6,25]]);
plotly_pie.draw();

colorString (from Subtle.COLORS, hex or rgb)COLORS['primary']Plot color
donutBooleanfalseDisplay as a donut
show_labelsBooleantrueShow labels on the plot
zindexNumber0Index to place the plot above or below other traces

Text

Use Subtle.PlotlyObject.add_text(x, y, text, params) to text to the plotly object.

The x, y and text arguments can be strings (if you want to add a single piece of text) or arrays (if you want to add multiple pieces of text).

paramtypedefaultdescription
textpositionString ("top", "middle", "bottom" + "left", "right", "center")"middle center"Position of the text
showlegendBooleanfalseShow in legend
zindexNumber9999Index to place the text above or below other traces
