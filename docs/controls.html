Text simple

Use Subtle.TextSimple(object, params) to display a single value as a label

Example

 var text_simple1 = new Subtle.TextSimple("text_simple1", {units:""});
var text_simple2 = new Subtle.TextSimple("text_simple2", {units:"L"});
var text_simple3 = new Subtle.TextSimple("text_simple3", {units:"kcal"});

text_simple1.update(250);
text_simple2.update(1.25684);
text_simple3.update(1250);

Params

paramtypedefaultdescription
valueNumber / String0Initial value
unitsString""Units
v_alignString ("top", "center", "bottom")"center"Vertical alignment
h_alignString ("left", "center", "right")"center"Horizontal alignment
roundingNumber2Decimal position for rounding
font_sizeNumber36Font size (in pixels)

Update

Historic Gauge

Use Subtle.GaugeAdv(object, params) to display a gauge that also shows a range of historical values.

Example

 var gaugeh = new Subtle.GaugeHistoric("gaugeh", {});
window.setInterval(function(){
    gaugeh.update(Math.random()*0.25+0.5);
}, 500);

Params

The historic gauge is derived from the normal gauge, and thus has all it's params, with differences on the following:

paramtypedefaultdescription
inner_paddingNumber3Makes the width of the gauge's bar thinner.
history_lengthNumber3600Determines for how long (in seconds) the gauge stores the values to display the range.

Update

Call the update method with the new value to update the control

Subtle.GaugeHistoric.update(value, time = "")

Here the time can be specified as an ISO string (YYYY-MM-DD HH:MM:SS) or as a javascript Date (the default value, an empty string, means that the value passed corresponds to the present time).

MultiTimePlot

Use Subtle.MultiTimePlot(object, params) to display small time plot with multiple series

Example

 var multitimeplot = new Subtle.MultiTimePlot("multitimeplot", 
    {limit_high:0.8, 
    limit_low:0.2, 
    show_labels:true,  
    shade_limits: true, 
    y_min: 0, 
    y_max: 1, 
    right_offset: 0}
);
 
var mt = 0;
window.setInterval(function(){
    multitimeplot.update({
        "A": Math.sin(mt*0.1)*0.2+0.5,
        "B": Math.sin(mt*0.1+2.1)*0.2+0.5,
        "C": Math.sin(mt*0.1+4.2)*0.2+0.5,
    });
    mt += 1;
}, 50);

Params

The MultiTimePlot inherits all params from TimePlot, with the following differences:

paramtypedefaultdescription
labelsArray[String]["A", "B", "C"]The labels for the time plot series
colorsArray[String]Subtle.COLOR_PALLETES["main"]An array with the colors for each of the series of the plot

Update

Call the update method with an object {serie: value} to update the control. You do not need to include all series, only the ones you want to update.

Subtle.MultiTimePlot.update({"A" : value, "B" : value, ...}, time = "")

Here the time can be specified as an ISO string (YYYY-MM-DD HH:MM:SS) or as a javascript Date (the default value, an empty string, means that the value passed corresponds to the present time).

LinePlot

Use Subtle.LinePlot(object, params) to display a line plot.

A LinePlot can be oriented horizontally or vertically, but in both cases the y values refer to the changing data and the x values refer to the index data (for example, if we want to represent the temperature at different dephts in a tank, the y values would be the temperature and the x values would be the depth).

Example

 var lineplot1 = new Subtle.LinePlot("lineplot1",
  {values_x : [0.1,0.5,0.9], 
  labels:["Alpha","Beta","Gamma"], 
  ticks_minor:[-0.5, 0.5], 
  show_labels:true, 
  show_values:true, show_ticks:true}
);

var lineplot2 = new Subtle.LinePlot("lineplot2",
  {direction:"vertical", 
  label_offset:100, 
  value_offset:40, 
  values_x : [0.1,0.5,0.9], 
  labels:["Alpha","Beta","Gamma"], 
  ticks_minor:[-0.5, 0.5], 
  show_labels:true, 
  show_values:true, 
  show_ticks:true}
);

lineplot1.update([0.1,0.5,0.3]);
lineplot2.update([0.1,0.5,0.3]);

Params

Params

paramtypedefaultdescription
imgString""Path to the svg file

stretchBooleantrueStretch the svg image size to fill the control
hover_effectBooleantrueChanges the color of the active items slighltly when you hover over them.

Update

You can add interactive elements to the svg such as tooltips and onclick events. To do so, the interactive elements must have a unique id (within the scope of the html document). The id of each element can be set by editing the xml code of the svg file (using any text editor), and you can group multiple elements using the <g></g> tag.

The following methods allow you to add interactive elements:

  1. Use SvgView.set_child_color(element_id, color) to change the color of an element.
  2. Use SvgView.set_child_text(element_id, text) to change the text of an element.
  3. Use SvgView.set_child_onclick(element_id, onclick) to set the onclick event on an element.
  4. Use SvgView.set_child_bind(element1_id1, element2_id) to bind two elements (so that the second element remains hidden until you hover on the first element).
  5. Use SvgView.set_child_tooltip(element_id, tooltip) to add a tooltip to an element (requires tippy.js and popper.js). The tooltip parameter can be html code.
  6. Use SvgView.set_child_for_html(element_id) to prepare an element to include other html controls. This is done using the foreignObject tag inside the svg. You have to be careful when creating your svg though, because any value of ViewBox that causes the svg to rescale will cause the elements inside the foreignObject to grow in size.

mymap.update_marker("new_york", {icon:"<i class='icon-pin'></i>", position: [40.7127, -74.0059], tooltip: "New York", icon_top: 24, icon_left: 12});

mymap2.update_marker("m2_icon", {icon:"<i class='icon-car-top'>></i>", position: [40.784188498795274, -73.9586093512171], tooltip: "Current vehicle location", icon_top: 12, icon_left: 12, rotation: -150});

Table

Use Subtle.Table(object, params) to display a table with data

Example

 var table = new Subtle.Table("table_div", 
  {headers: ["Row", "North", "East", "South", "West"], 
  allow_sort: true, 
  insert_on_top: true}
);
 
table.update("row 1", {"Row": 1, "North": 10});
table.update("row 2", {"Row": 2, "North": 9, "East": 8});
table.update("row 3", {"Row": 3, "North": 9, "South": 12});
table.update("row 4", {"Row": 4, "North": 10, "East": 55, "South": 3, "West": 15});
 
table.update("row 2", {"East": 35});

Params

paramtypedefaultdescription
headersArray[String]["A","B","C","D"]Array with the names of each column
header_aliasesObject ({header: alias}){}A map to change the header's names on the table
table_classString"subtletable"CSS class to apply to the table for styling
show_row_headersBooleantrueShow a column that indicates the row number
row_header_widthNumber30Row heades' column width (in pixels)

remove_old_dataBooleantrueRemove old rows when the number of rows passes the limit (max_displayed_data)

Update

Call the update method to add new rows to the table:

Subtle.Table.update(id, data)

For each row you insert, you need to specify a row id. You you also pass an empty string ("") as id and the data will be assigned an auto id.

The data argument on the update function is a dict {column: value}, where column is the name of the column as defined by the headers (Table.params.headers)

Tree View

UseSubtle.TreeView(object, nodes, params) to display a tree view

The nodes argument is a list with all the nodes in the tree (see the example).

Example

var nodes = [
    { path: 'A/B/X' },
    { path: 'A/B/Y' },
    { path: 'A/C/X' },
    { path: 'A/C/Z' },
    { path: 'A/W' },
];
var treeview = new Subtle.TreeView("treeview_div", nodes, {show_arrows: true, show_checkboxes: true})

Params

Update

The path of the current node is stored in treeview.current_node

To get the checkbox states of all nodes, check treeview.nodes_checkbox_state

Additionally, the following methods are available:

treeview.expand_all()

treeview.collapse_all()

treeview.search(text) // filter the nodes by text

For each row you insert, you need to specify a row id. You you also pass an empty string ("") as id and the data will be assigned an auto id.

The data argument on the update function is a dict {column: value}, where column is the name of the column as defined by the headers (Table.params.headers)

If you want to change the nodes of the tree, you can edit the treeview.nodes object and then call the treeview.draw() method.
