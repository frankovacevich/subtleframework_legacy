Subtleframe

Subtle free is a free and open source html, css and javascript framework that makes building dashboards and IOT GUI applications easier.

Subtleframe consists of a css and a js files that can be found on the github site.

Some parts of subtleframe require some dependencies, like Popper.js. The required dependencies are specified on the documentation.

Subtleframe works well with Bootstrap.
